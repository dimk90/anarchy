# Nerd Font

Nerd Fonts contains a high number of glyphs (icons) from popular "iconic fonts" 
such as Font Awesome, Devicons, Octicons, and others.
Useful for terminal, text editor, or IDE.


## Usage

The `Symbols Nerd Font` contains only icons without alphanumeric characters.
So it should be used in combination with another font.  

VSCode font settings:
```json
{
    // ...

    "terminal.integrated.fontFamily": "Iosevka Term, Symbols Nerd Font",

    // ...
}
```
